# Asteroidit

Sovellus on peli, jossa käyttäjä ohjaa avaruusalusta, ja tavoitteena on tuhota tielle osuneet asteroidit.

## Asennus

1. `poetry install` asentaa sovelluksen riippuvuudet
2. `poetry run invoke start` käynnistää sovelluksen
